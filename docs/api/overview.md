# API Reference - Обзор

Этот раздел содержит полную документацию API Oasis 3D.

## Структура API

API Oasis 3D организован в следующие модули:

- **Core API** — Основные функции и классы
- **Components** — Компоненты системы
- **Utilities** — Вспомогательные функции

## Основные модули

### Core Module

Основной модуль, содержащий базовые классы и функции.

```python
import 3doasis.core
```

### Components Module

Модуль компонентов для расширения функциональности.

```python
import 3doasis.components
```

### Utilities Module

Вспомогательные утилиты.

```python
import 3doasis.utils
```

## Соглашения

### Типы данных

- `str` — Строка
- `int` — Целое число
- `float` — Число с плавающей точкой
- `bool` — Булево значение
- `dict` — Словарь
- `list` — Список

### Обработка ошибок

Все функции API могут выбрасывать исключения. См. документацию конкретных функций для деталей.
